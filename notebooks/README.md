# Notebooks

這個資料夾只保留目前仍屬正式流程或常用工具的 notebook。

## Core

- `compile_tax_notes.ipynb`
  合併同一門課的多個 Markdown 課時，輸出單一合併 Markdown，並可進一步輸出 Word。

- `md-fix.ipynb`
  批次整理 Markdown markup 規則，例如 front matter 後空行、正文 `---` 分隔線、標題後空行、清單前後空行、條文引用格式等。

## Optional

- `markdown_export_toolkit.ipynb`
  將單一 Markdown 穩定輸出成 Word/Markdown，偏向單檔匯出工具，不是課程合併主流程。
